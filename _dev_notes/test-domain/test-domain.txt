There is a docker image availabe for bind9 that is used to test domains locally
with local dns server.

the docker is found in scripts/_external/

You will need to configure local files on your machine for bind.
See here for example:

etc/bind/named.conf.local:
-
zone "grascannabis.org" {
  type master;
  file "./zones/grascannabis.org";
};

etc/bind/zones/grascannabis.org:
-
;
@               IN      A       127.0.0.1:3000
