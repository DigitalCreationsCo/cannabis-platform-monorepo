# Testing Pod Connectivity
If you need to test connectivity inside the kubernetes cluster, you can run a debugging pod.
```
$ kubectl run -it --rm debug-pod --image=alpine --restart=Never -- sh
```
sh into the debugging pod, and curl to your pod that you're troubleshooting to test a connection.
apk update && apk add curl
curl <service-name>:<port>/<path> 
or 
curl <internal-ip>/<path>
